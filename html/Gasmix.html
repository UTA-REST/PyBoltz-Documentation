Gasmix module¶

class Gasmix.Gasmix¶

Bases: object

The Gasmix object is used to coordinate the calling of different gas functions. It contains an array of six Gas object structs. This object is used by the Mixer functions to get the cross section outputs.

ExtraParameters¶

Dictionary used to pass extra parameters into the gas functions.

Gases¶

Array of six gases used to store the input and output of the gas functions. Uses the Gas struct.

InitWithInfo()¶

This functions simply initiates the gas data from the parameters. This functions fills the output arrays to zeros.

Run()¶

This functions calls the corresponding gas functions.

reset()¶

Function used to zero out the main output arrays.
