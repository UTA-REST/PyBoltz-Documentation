Magboltz module¶

class Magboltz.Magboltz¶

Bases: object

This is the main object used to start the simulation, as well as store the information of the simulation. It has most of the needed arrays, and variables.

Start()¶

This is the main function that starts the magboltz simulation/calculation.

The simulation starts by calculating the momentum cross section values for the requested gas mixture. If EFINAL is equal to 0.0 it will then keep calling the ELIMIT functions and the MIXER functions to find the electron Integration limit.

Finally Magboltz calls the Monte carlo functions, which is where the main simulation happens. The outputs are stored in the the parent object of this function.

For more info on the main output variables check the git repository readme: PyBoltz repository
