EnergyLimits module¶

EnergyLimits.EnergyLimit()¶

This function is used to calculate the upper electron energy limit by simulating the collisions. If it crosses the Object.FinalEnergy value it will set self.IELOW to 1 which would get the PyBoltz object to try a higher FinalEnergy value. This function is used when there is no magnetic field.

The test is carried out for a sample of collisions that are smaller than the full sample by a factor of 1/isamp

Electric field in z direction.

The object parameter is the PyBoltz object to be setup and used in the simulation.
